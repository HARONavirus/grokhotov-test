# Тестовое задание для компании https://grokhotov.ru

## Как запустить?

1. Устнаовить LTS версию Node.JS
2. Установить редактор кода (например VSCode)
3. Склонировать репозиторий к себе на ПК
4. `npm install`
5. `npm run dev`
6. Готово!

**⚙️ Стэк технологий:**

- Nuxt 3
- Pinia
- Axios
- Swiper.js
- Composition API / Options API
